# badminton-rally-tracke

## 概要
badminton-rally-trackeは、バドミントンのラリーを追跡・分析するためのアプリケーションです。試合中の各ショットや得点、選手の動きを記録し、戦術やパフォーマンス分析をサポートします。

## 主な特徴
- ショット履歴の記録と表示
- 得点管理とヒートマップ生成
- プレイヤーごとの統計データ分析
- 分析結果のエクスポート機能

## インストール方法

1. リポジトリをクローンします：

   ```bash
   git clone https://github.com/tbw0619/badminton-rally-tracke.git
   ```
2. 必要なライブラリをインストールします：

   ```bash
   pip install -r requirements.txt
   ```

## 使い方

1. アプリを起動します：
   ```bash
   python main.py
   ```
2. 画面の指示に従って、ラリー情報やプレイヤー情報を入力してください。
3. 記録を保存し、分析結果を確認できます。
